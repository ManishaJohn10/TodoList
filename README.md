TODO LIST
A simple and elegant full-stack Todo List application built using HTML, CSS, JavaScript, Express.js, MongoDB, and EJS.
The app lets users add tasks, delete tasks, and reset the entire list,all stored in a MongoDB database.
Features include:Add new tasks,Delete tasks instantly,Auto-submit on checkbox,Clear all tasks at once,
Persistent storage using MongoDB,Clean and aesthetic UI using CSS and EJS.
Tech Stack includes:HTML,CSS,EJS(Frontend) Node.js,Express.js(Backend) MongoDB,Mongoose(Database)
