## TODO LIST

A clean and simple To-Do List application built using Node.js, Express.js, MongoDB, and EJS, with a modern UI designed using CSS.The app allows users to add tasks, delete tasks, and clear all tasks with data stored permanently in MongoDB.

## Features

1. Add new tasks
2. Check off & Delete tasks instantly
3. Clear all tasks
4. Persistent storage using MongoDB
5. Aesthetic and minimal UI with CSS and EJS

## Tech Stack

Frontend:
1. HTML
2. CSS
3. EJS
   
Backend:
1. Node.js
2. Express.js

Database:
1. MongoDB
2. Mongoose

# Installation and Local Setup

Clone the repository:
```bash
git clone https://github.com/your-username/todolist-app.git
cd todolist-app
```
Install Dependencies:
```bash
npm install
```
Create a ```.env ``` File:
```bash
MONGO_URI=your_mongodb_connection_string
```
Run the server:
```bash
npm start
```
Open in browser:
http://localhost:3000
